# Standards & Correlations Reference

This page lists the industry standards, correlations, and equations used throughout the Well Testing Suite.

## Metering

| Module | Standard / Method | Reference |
|---|---|---|
| AGA-3 Gas Rate | AGA Report No. 3 / API MPMS 14.3 | Orifice meter gas flow calculation |
| AGA-3 Discharge Coefficient | Reader-Harris/Gallagher (RHG) equation | Iterative Cd with Reynolds number |
| AGA-3 Z-Factor | Hall-Yarborough correlation | Compressibility factor from Tpr, Ppr |
| AGA-3 Mixing Rules | Piper-McCain-Corredor | Pseudo-critical properties with CO2/H2S |
| Turbine Meter | API MPMS Chapter 5.3 | Liquid turbine meter sizing and validation |

## Reservoir & Production

| Module | Standard / Method | Reference |
|---|---|---|
| DCA — Exponential/Hyperbolic/Harmonic | Arps (1945) | J.J. Arps, "Analysis of Decline Curves", Trans. AIME |
| DCA — Duong | Duong (2011) | A.N. Duong, "Rate-Decline Analysis for Fracture-Dominated Shale Reservoirs" |
| DCA — Stretched Exponential | Valko & Assadi (2010) | SEPD model for unconventional reservoirs |
| PTA — Horner | Horner (1951) | Buildup test analysis, semi-log straight line |
| PTA — Bourdet Derivative | Bourdet et al. (1989) | Log-log pressure derivative diagnostic |
| PTA — Superposition | Multi-rate superposition time | Rate-normalised analysis |
| Solution GOR — Standing | Standing (1947) | Rs from pressure, temperature, API, gas SG |
| Solution GOR — Vasquez & Beggs | Vasquez & Beggs (1980) | Rs correlation with API gravity groups |
| Bubble Point | Standing (1947) | Bubble point pressure from Rs, T, SG, API |
| Shrinkage Factor | Standing (1947) | Oil formation volume factor (Bo) |
| Fluid Properties — API Correction | API MPMS Chapter 11.1 | Temperature correction to 60°F |
