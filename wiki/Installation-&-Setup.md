# Installation & Setup

## Requirements

- A modern web browser (Chrome, Firefox, Edge, or Safari)
- No internet connection required
- No installation or build step

## Getting Started

1. **Download** or clone the repository.
2. **Open** `well-testing-app.html` directly in your browser (double-click the file, or use `File > Open`).
3. The application loads instantly — no server, no compilation, no dependencies.

## First Steps

1. Navigate to **Client & Well Information** (loaded by default on first visit).
2. Fill in your client, well, and engineer details — these appear on all PDF exports.
3. Optionally upload a **client logo** for co-branded report cover pages.
4. Select any calculator from the sidebar to begin.

## Data Storage

All input values are automatically saved to your browser's `localStorage`. This means:

- Your data persists between sessions (closing and reopening the browser).
- Clearing browser data (cache/cookies) will erase saved values.
- Different browsers maintain separate storage (Chrome and Firefox don't share data).
- Use **Export Session** to create a portable backup file.

## Deployment

To deploy for a team:

- Host the single HTML file on any web server, intranet, or shared drive.
- No backend, database, or API is needed.
- Each user's data stays in their own browser.
