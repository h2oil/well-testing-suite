# Field Calculators

Quick-reference calculators for everyday oilfield engineering tasks.

## Gas Calculations

| Calculator | Description |
|---|---|
| Gas Velocity | Velocity of gas in a pipe given flow rate, pressure, temperature, and pipe ID |
| Gas Volume | Quick gas volume at standard/actual conditions |
| Gas Gradient | Gas pressure gradient from specific gravity and depth |
| Downhole Pressure | Simple gas gradient-based downhole pressure estimate |

## Fluid Properties

| Calculator | Description |
|---|---|
| API Gravity Correction | Correct observed API gravity and density to standard 60°F |
| SG to API / API to SG | Bidirectional conversion between specific gravity and API gravity |
| Bubble Point | Estimate bubble point pressure using Standing correlation |
| Shrinkage Factor | Estimate oil shrinkage from separator to stock tank using Standing correlation |

## Tank Calculator

| Calculator | Description |
|---|---|
| Rectangular Tank | Volume from length, width, and height |
| Cylindrical Tank | Volume for horizontal or vertical cylindrical vessels |
| Tank Flow Rate | Rate calculated from change in tank level over a time interval |
| Tank Weight | Estimate fluid weight from tank volume and specific gravity |

## Electrical & Pumps

| Calculator | Description |
|---|---|
| 3-Phase Motor Current | Full-load current from kW, voltage, power factor, and efficiency |
| kW to HP / HP to kW | Bidirectional power unit conversion |
| Mud Pump Output | Pump output (gpm and bbl/min) from liner size, stroke length, SPM, and efficiency |

## Chemical & Dosage

| Calculator | Description |
|---|---|
| Chemical Injection Rate | Required injection rate (L/hr) for multiple chemicals based on production flow and target ppm |
| Chloride Determination | Argentometric (silver nitrate titration) method for chloride concentration in produced water |

## Separator Retention Time

Estimate fluid retention time in a separator vessel from dimensions, normal liquid level, and flow rate.

## Bottoms Up Time

Calculate circulation time for fluid returns based on pump rate and annular/tubing volume.

## Solution GOR

Estimate solution gas-oil ratio using Standing and Vasquez & Beggs correlations.

## Oil & Gas Rate + GOR

Combined oil rate, gas rate, GOR, and condensate ratio calculation with volume correction factor (VCF).

## MCF & Shrinkage %

Meter Correction Factor and shrinkage percentage from turbine meter and surge tank readings.

## Casing & Tubing

Casing/tubing dimensions, annular capacity (bbl/ft), and total annular volume from measured depth.

## Choke Conversions

Convert choke sizes between 64ths of an inch, decimal inches, and millimeters.

## Analog Signal Conversions

Convert between 4-20 mA, 0-10 V, and process values for instrumentation calibration and troubleshooting.
